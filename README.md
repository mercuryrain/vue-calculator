# Calculator

The simple calculator implemented in Vue / Vuex.  

Feature list: 
* basic arithmetic operations
* percentage
* EUR <=> USD conversion
* behavior of "=" operator mocks the behavior of hardware calculators
* Offline mode (PWA)
* Unit and e2e tests
